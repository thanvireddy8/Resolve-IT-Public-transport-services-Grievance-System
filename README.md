# 🚍 Resolve-It: Public Transport Grievance System

## 📌 Overview
Resolve-It is a full-stack web application designed to manage and track public transport complaints efficiently.  
It allows users to raise complaints, track status, and provide feedback, while admins manage and resolve issues.

---

## 🚀 Features

### 👤 User Side
- User Registration & Login  
- Raise Complaint  
- Track Complaint Status  
- View Complaint History  
- Feedback System  
- Notifications  

### 🛠️ Admin Side
- Dashboard Overview (Total, Pending, Resolved)  
- Manage Complaints  
- Update Status  
- User Management  
- Reports & Analytics  

---

## ✨ Key Highlights
- Real-time complaint tracking system  
- Role-based access (User/Admin)  
- AI Chatbot for smart assistance  
- Complaint analytics and reporting  
- Scalable full-stack architecture  

---

## 🤖 AI Chatbot Feature
- Suggests complaint category based on user input  
- Guides users through complaint process  
- Redirects users directly to complaint form  
- Improves user experience and reduces manual effort
